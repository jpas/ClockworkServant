# Spell: Invisibility, Greater

^([PRPG Core Rulebook][ss-greater-invisibility] pg. 302 | PFS Legal | [Archives of Nehtys][sn-greater-invisibility])

**School** illusion (glamer); **Level** alchemist 4, antipaladin 4, arcanist 4, bard 4, inquisitor 4, investigator 4, magus 4, red mantis assassin 4, skald 4, sorcerer/wizard 4, summoner 3

### Casting

**Casting Time** 1 standard action  

**Components** V, S

### Effect

**Range** personal or touch  

**Target** you or creature touched  

**Duration** 1 round/level (D)  

**Saving Throw** Will negates (harmless); **Spell Resistance** yes (harmless)

### Description

This spell functions like _[invisibility]_, except that it doesn't end if the subject attacks.
